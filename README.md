# Progg Camp
## Zaawansowane interfejsy użytkownika - Projekt semestralny 2026
### Autor: Wiktor Wojtanowski

## Opis projektu
Progg Camp to aplikacja webowa do nauki programowania dla początkujących. Strona oferuje kursy z następujących języków programowania:
* Python
* HTML
* CSS

Pod koniec każdego kursu należy zaliczyć test składający z 15-20 pytań. Po osiągnięciu min. 50% następuje zaliczenie kursu.
