# Progg Camp
## Zaawansowane interfejsy użytkownika - Projekt semestralny 2026
### Autor: Wiktor Wojtanowski

## Opis projektu
Progg Camp to aplikacja webowa do nauki programowania dla początkujących. Strona oferuje kursy z następujących języków programowania:
* Python
* HTML
* CSS

Pod koniec każdego kursu należy zaliczyć test składający z 15-20 pytań. Po osiągnięciu min. 50% następuje zaliczenie kursu.

## Instrukacja uruchomienia
```bash
npm install
npm run build
npm run dev
```

## Użyte technologie
* React 18.3 - Główny framework do warstwy Frontend
* Vite 8.0 - Lokalny serwer deweloperski
* Chakra UI v3 - Główna biblioteka UI
* React Hook Form - Walidacja formularzy
* Zustand - Zarządzanie stanem globalnym
* React Markdown - Wyświetlanie tekstu w formacie Markdown
* Framer Motion - Animacje przejścia pomiędzy stronami
* Lucide React - Ikony 
* Chrome DevTools - Testowanie i debuggowanie
