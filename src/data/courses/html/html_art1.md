# 💻 Rozdział 1: Pobieranie i przygotowanie narzędzi

Witamy na pokładzie! Zanim zanurzymy się w świat tagów HTML i struktury stron internetowych, musimy założyć nasz „Warsztat Programisty”. Nie piszemy kodu długopisem i papierem – potrzebujemy specjalnego programu, który pomoże nam pisać kod czysto, czytelnie i bez błędów.

Ten rozdział jest bardzo techniczny, ale obiecuję Ci: **to jest super proste!** Po jego zakończeniu będziesz gotowy do pisania pierwszego kodu HTML.

---

## 🛠️ Co to jest edytor kodu? (IDE vs. Editor)

Zanim przejdziemy do instalacji, musimy wiedzieć, co dokładnie pobieramy.

*   **Edytor Kodu (Code Editor):** To specjalny program zaprojektowany *tylko* do pisania kodu. Nie jest to zwykły edytor tekstu (jak Notatnik), ponieważ rozróżnia kod od normalnego tekstu, podświetla składnię (kolory dla różnych elementów) i pomaga nam zapamiętać skróty klawiszowe.
*   **VS Code (Visual Studio Code):** To obecnie najbardziej popularny i najlepszy edytor kodu na rynku. Jest darmowy i potężny.

> 💡 **Zapamiętaj:** VS Code to nasz główny „dzbanek” do pisania kodu.

## 📥 Krok 1: Pobieranie Visual Studio Code

Pobieramy najnowszą wersję edytora. Proces zajmie Ci mniej czasu, niż zrobienie filiżanki herbaty! 🫖

1.  **Wejdź na oficjalną stronę:**
    [Visual Studio Code Download)](https://code.visualstudio.com/)
2.  **Wybierz odpowiednią wersję:** Strona automatycznie wykryje system operacyjny, na którym pracujesz (Windows, macOS czy Linux). Kliknij przycisk pobrania dla swojego systemu.
3.  **Uruchom instalator:** Po zakończeniu pobierania znajdź plik i uruchom go. Postępuj zgodnie z instrukcjami kreatora.

### 🖥️ Instalacja na Windowsie (Kluczowe kroki!)

Jeśli używasz systemu Windows, zwróć szczególną uwagę na ostatni ekran instalatora:

*   **Zaznacz opcje:** Upewnij się, że zaznaczyłeś opcję typu *„Add to PATH”* lub *„Open with Code from terminal”*. Te dodatkowe akcesoria ułatwią nam pracę później z wiersza poleceń.
*   Kliknij **Finish**.

### 🍎 Instalacja na macOS (Kluczowe kroki!)

1.  Pociągnij ikonę VS Code do folderu `Applications`.
2.  Otwórz i uruchom program.

## ✨ Krok 2: Pierwsze uruchomienie i konfiguracja podstawowa

Po zainstalowaniu edytora, otwórz go! Powinieneś zobaczyć czystą, minimalistyczną przestrzeń do pisania kodu.

### A. Tworzenie pierwszego projektu (Folder)

Zaleca się, aby wszystkie projekty były przechowywane w dedykowanym folderze.

1.  Utwórz na pulpicie nowy folder i nazwij go: `Moje_Pierwsze_Strony`.
2.  W VS Code kliknij **File > Open Folder...** (Plik > Otwórz folder...).
3.  Wskaż folder `Moje_Pierwsze_Strony` i otwórz go w edytorze.

### B. Tworzenie pliku HTML

Każda strona internetowa musi być zapisana jako plik z rozszerzeniem `.html`.

1.  W lewym górnym rogu VS Code (panel Eksplorator) kliknij ikonę **Nowy Plik** (zazwyczaj jest to ikona arkusza papieru).
2.  Nazwij ten nowy plik: `index.html` (To zawsze będzie nasz główny, pierwszy plik!).

### C. Testowanie magicznego triku ✨ (Emmet)

VS Code ma wbudowaną funkcję super-pomocniczą dla HTML! Możesz stworzyć pełną strukturę strony za pomocą kilku liter.

1.  Upewnij się, że kursor znajduje się w pliku `index.html`.
2.  Napisz: `!` (Wykrzyknik).
3.  Naciśnij **Tab** lub **Enter**.

💥 *Co się stało?* VS Code automatycznie wypełni cały boilerplate kodu HTML5! To znak, że wszystko działa poprawnie.
